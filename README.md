This project is a simple Python application that detects faces in an image and automatically applies a blur effect to them.
The goal of the project is to demonstrate how computer vision techniques can be used to protect privacy by hiding facial identities.
The program uses a face detection algorithm to locate faces in each frame and then applies a blur filter over the detected regions. 
This creates an anonymization effect while keeping the rest of the image visible.

Technologies Used:
Python
OpenCV

Features:
Detects human faces in real time or in images
Automatically blurs detected faces
Simple and lightweight implementation for learning computer vision basics

Purpose:
This project was created as a small experiment in computer vision and image processing, exploring how algorithms can be used to automatically anonymize visual data.
